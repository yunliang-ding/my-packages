## 安装

```js
pnpm i
```

## 启动

```
npm start
```

## 部署文档

```
npm run docs:build
```

## 打包

```
cd packages/xxx

npm run build
```

## 前端生态包

- @yl-d/design 通用 ui 库
- @yl-d/hooks 状态管理库
- @yl-d/shared 工具库
- @yl-d/code-editor 代码编辑组件
- @yl-d/low-code 低代码相关
- @yl-d/cli 工程脚手架
- @yl-d/docs 组件库文档脚手架
- @yl-d/create-cli 快速选择项目模版
